2023

Tamagochi

This project emulates the popular Tamagochis of the 90s. The pet to take care of here is a Bulbasaur from the Pokemon franchise.
You can feed and play with it. It also has some systems running in the background for the boredom and health values to change over time.
By interacting with Bulbasaur you can earn coins which then can be spent for food and medicine, in case it should be needed.

This program comes without working UI and is completely text-based.

Password Generator

This tool creates passwords for the user. It is possible to declare the length of the password as well as if it should contain upper case, lower case letters, symbols and/or numbers.
The generated password can then be copied to the clipboard by button press.

The password generator has a fully functional UI.
